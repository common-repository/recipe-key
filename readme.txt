Recipe Key adds icons to your posts by integrating with your categories and tags. Recipe Key increases sites navigation, site structure and category page viewability by providing site visitors with a quick and simple visual with links to similar content. 

== Description ==
<strong>A bloggers dream come true!</strong>

Recipe Key adds visual icons to your posts helping your site visitors find more of what interests them. It does this by integrating with your categories and tags. Use illustrated dietary icons, or abbreviated icons (e.g. GF for gluten free). Great for food, travel or DIY bloggers.

== Frequently Asked Questions ==
= I've Installed The Plugin Now What? =

If you don't already have category and/or tags, it's time to add them. For example, add the category or tag 'gluten free' or 'kid friendly' and that icon will show up on any post labeled as 'gluten free' or 'kid friendly'.

= Icons aren't showing up, why? =

Verify that you have categories or tags in Wordpress that match the configuration in Recipe Key. You can use the Recipe Key admin page under the admin dashboard > settings > Recipe Key > Summary to see why Recipe Key isn't picking up the associated tags or categories. Make sure you've entered your wordpress category/tags into Recipe Key exactly how they appear, they are case sensitive. 

= Can I change recipe key colors? =

Yes! You can change change the style and colors of recipe key within the settings.

= Do I need to use the dietary illustrated icons? =

Nope, you can remove the illistrated Icons and use Recipe Key for any type of website (e.g. Travel, DIY, Fitness, etc).

= Can I use recipe key with Gutenburg? =

Yes, Recipe Key works with both the Classic and Gutenberg editors.

= Can Recipe Key be used with any language? =

Yes! Because Recipe Keyâ€™s category and tags are 100% customizable, you can use any language.
